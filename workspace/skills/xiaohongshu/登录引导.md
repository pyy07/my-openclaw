# 小红书 MCP 登录引导

## 一、先安装二进制（仅需一次）

在终端执行（在**本机**执行，网络需能访问 GitHub）：

```bash
cd /Users/lukepan/.openclaw/workspace/skills/xiaohongshu/scripts
chmod +x install-xiaohongshu-mcp.sh
./install-xiaohongshu-mcp.sh
```

- 脚本会自动下载适合你 Mac 的版本（Apple Silicon 用 arm64），并安装到 `~/.local/bin`。
- 若下载超时，脚本会提示你**手动下载**：打开 [xiaohongshu-mcp Releases](https://github.com/xpzouying/xiaohongshu-mcp/releases)，下载 `xiaohongshu-mcp-darwin-arm64.tar.gz`（M1/M2/M3）或 `darwin-amd64`（Intel），解压后把两个可执行文件放到 `~/.local/bin` 并改名为 `xiaohongshu-mcp`、`xiaohongshu-login`，再 `chmod +x`。

确保 PATH 里有 `~/.local/bin`（很多系统默认已有）：

```bash
export PATH="$HOME/.local/bin:$PATH"
# 可写入 ~/.zshrc 以便长期生效
```

验证安装：

```bash
cd /Users/lukepan/.openclaw/workspace/skills/xiaohongshu/scripts
./install-check.sh
```

看到「所有依赖已就绪」即可进行下一步。

---

## 二、登录小红书账号

1. **在终端执行**（需本机有图形界面，会弹浏览器或二维码）：

   ```bash
   cd /Users/lukepan/.openclaw/workspace/skills/xiaohongshu/scripts
   ./login.sh
   ```

2. **按提示操作**：
   - 会启动 **xiaohongshu-login** 程序；
   - 若弹出**浏览器**：在浏览器里用你的小红书账号登录；
   - 若出现**二维码**：用手机打开**小红书 App**，扫码登录。

3. **登录成功后**：
   - 程序会把 Cookie 保存到本机（常见路径如 `~/.xiaohongshu/` 或 `/tmp/cookies.json`）；
   - 之后 MCP 发帖、评论等都会用这个账号。

4. **若在无图形界面的服务器上**：
   - 需在**有桌面的电脑**上运行一次 `~/.local/bin/xiaohongshu-login` 完成登录；
   - 把生成的 cookies 文件拷到服务器的 `~/.xiaohongshu/cookies.json`，再在服务器上启动 MCP。

---

## 三、启动 MCP 服务（发帖/评论前要开着）

登录完成后，需要**先启动 MCP 服务**，OpenClaw 或脚本才能调用发帖、搜索等能力：

```bash
cd /Users/lukepan/.openclaw/workspace/skills/xiaohongshu/scripts
./start-mcp.sh
```

- 服务地址：`http://localhost:18060/mcp`
- 关闭服务：`./stop-mcp.sh`
- 查看登录状态：`./status.sh`

---

## 四、小结

| 步骤 | 命令 |
|------|------|
| 1. 安装二进制 | `./install-xiaohongshu-mcp.sh` |
| 2. 检查依赖 | `./install-check.sh` |
| 3. 登录账号 | `./login.sh`（浏览器或 App 扫码） |
| 4. 启动服务 | `./start-mcp.sh` |

完成后即可在飞书里让 OpenClaw「发一条小红书」或使用搜索、评论等能力。
